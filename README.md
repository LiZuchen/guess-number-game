# 🎯 猜数字游戏

一个简单有趣的两人猜数字对战游戏。

## 💡 起源

两个人无聊时，没有好的电子设备，又不想一直刷手机，就想找个简单又能互动的游戏。于是有了这个猜数字游戏——只需要一个能打开网页的设备，就能玩上好久。

## 📖 游戏规则

1. **设置阶段**：两位玩家各自秘密输入一个四位数（1000-9999）
2. **猜测阶段**：两人轮流猜测对方的数字
3. **反馈**：每次猜测会告诉你对了几个数字（位置也要对）
4. **胜利**：先猜中对方全部 4 个数字的玩家获胜！

### 示例
- A 选了 `2331`
- B 猜 `3333` → 反馈：对了 2 个数字（第 2、3 位正确）
- B 猜 `2331` → 反馈：对了 4 个数字，B 获胜！🎉

## 🎮 如何玩

1. 玩家 1 输入一个四位数并确认
2. 玩家 2 输入一个四位数并确认
3. 点击"🎮 开始游戏"按钮
4. 玩家 1 先猜，轮流猜测对方的数字
5. 每次会反馈对了几个数字（位置也要对）
6. 先猜中 4 个数字者获胜！
7. 点击"再来一局"重新开始

## ✨ 特性

- 🎨 精美的渐变 UI 设计
- 📱 响应式布局，支持手机和电脑
- 🎉 胜利时的彩带特效
- 📜 完整的猜测历史记录
- ⌨️ 支持回车键快速提交
